pygame.pixelcopy
    pygame module for general pixel array copying
      pygame.pixelcopy.surface_to_array - copy surface pixels to an array objectcopy surface pixels to an array object
      pygame.pixelcopy.array_to_surface - copy an array object to a surfacecopy an array object to a surface
      pygame.pixelcopy.map_array - copy an array to another array, using surface formatcopy an array to another array, using surface format
      pygame.pixelcopy.make_surface - return a new surface with pixels copied from an arrayreturn a new surface with pixels copied from an array

    The pygame.pixelcopy module contains methods for copying between surfaces and objects exporting an array structure interface.
